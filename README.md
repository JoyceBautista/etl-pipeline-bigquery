# Pipeline ETL con Python y BigQuery

## Descripción
Pipeline ETL que extrae datos desde una API REST (DummyJSON), transforma datos JSON a formato tabular utilizando Pandas y los carga a Google BigQuery.

## Tecnologías
- Python
- Pandas
- Google BigQuery
- APIs REST

## Flujo
1. Extracción de datos desde API
2. Transformación con json_normalize
3. Limpieza de columnas
4. Carga a BigQuery
